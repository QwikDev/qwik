# Create Qwik ⚡️

## Interactive mode

```
npm create qwik@latest
```

## Command mode

```
npm create qwik@latest <starter> <projectName>
```

## API

```javascript
const { createApp } = require('create-qwik');

const opts = {
  projectName: 'my-project',
  starterId: 'todo',
  outDir: '/path/to/output/dir',
};

const result = await createApp(opts);
console.log(result);
```
