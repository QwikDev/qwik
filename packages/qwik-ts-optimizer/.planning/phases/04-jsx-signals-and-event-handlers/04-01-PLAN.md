  <action>
    Create `src/optimizer/jsx-transform.ts` with these exports:

    1. `classifyProp(exprNode: any, importedNames: Set<string>): 'const' | 'var'` — Analyze an expression AST node to determine if it's immutable or mutable. Immutability rules (per D-JSX-02 from RESEARCH.md):
       - CONST: string/number/boolean/null/undefined literals, template literals without expressions, imported identifiers (in importedNames set), member expressions where object is imported (`dep.thing`), CSS module refs, ternary with all-const operands, binary with all-const operands, `_wrapProp()` calls, `_fnSignal()` calls
       - VAR: global variables (not in importedNames, not declared locally), `window.*` / `document.*` / `globalThis.*`, function calls with unknown side effects, `signal.value` (MemberExpression with `.value`), any expression containing a VAR sub-expression

    2. `computeFlags(hasVarProps: boolean, childrenType: 'none' | 'static' | 'dynamic', inLoop: boolean): number` — Compute the flags bitmask:
       - Bit 0 (1): children type (1 if static/text children)
       - Bit 1 (2): props mutability (2 if NO varProps — i.e., all const)
       - Bit 2 (4): loop context
       - Combine: `3` = fully immutable (no varProps + const children), `1` = dynamic children only, `2` = has varProps, `0` = spread/unknown, `4` = loop, `6` = loop + no varProps

    3. `JsxKeyCounter` class — Per-module counter generating `"u6_0"`, `"u6_1"`, etc. Has `next(): string` method.

    4. `transformJsxElement(node: any, source: string, s: MagicString, importedNames: Set<string>, keyCounter: JsxKeyCounter): JsxTransformResult` — Transform a single JSX element node:
       - Extract tag name (string literal for HTML elements, identifier for components)
       - Iterate JSXAttribute nodes, classify each prop value
       - Build varProps object literal string (or `null`)
       - Build constProps object literal string (or `null`)
       - Process children (text -> string, expressions -> inline, multiple -> array)
       - Compute flags
       - Generate key (from explicit `key` prop or auto `u6_N`)
       - Return the `_jsxSorted(tag, varProps, constProps, children, flags, key)` call string

    Use magic-string for the replacement: `s.overwrite(node.start, node.end, jsxSortedCall)`.

    Handle JSXExpressionContainer nodes in children and props. Handle JSXText by trimming and quoting.

    Do NOT handle signal wrapping, event handler naming, or bind syntax in this task — those are separate modules that will modify props before/after this transform.
  </action>
